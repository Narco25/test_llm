01 — Architecture

Every component,
from scratch.

01
BPE Tokenizer

Byte-Pair Encoding built from raw bytes. Iteratively merges the most frequent adjacent pairs until the target vocabulary size is reached.

02
RMSNorm

Root Mean Square normalisation — pre-norm residual connections for stable training, no mean-centering needed.

03
RoPE

Rotary Position Embeddings encode position by rotating pairs of dimensions. No learned parameters, better length extrapolation.

04
Causal Attention

Multi-head self-attention with a causal mask. Each token attends only to positions ≤ its own, enabling autoregressive generation.

05
SwiGLU FFN

Feed-forward network with SwiGLU activation: silu(W₁x) ⊙ W₂x · W₃. Outperforms ReLU/GELU in practice.

06
Weight Tying

Input token embeddings and the output projection share weights. Halves parameter count and regularises the model.

02 — Quickstart

Up and running
in three commands.

# 1. Clone the repo
git clone https://github.com/YOUR_USERNAME/YOUR_REPO.git
cd YOUR_REPO

# 2. Install the only dependency
pip install torch

# 3. Run the built-in demo (trains + generates)
python llm_from_scratch.py --demo
# Train on your own text file
python llm_from_scratch.py \
  --train corpus.txt \
  --epochs 10 \
  --out model.pt

# Generate text from a trained model
python llm_from_scratch.py \
  --generate "Once upon a time" \
  --model model.pt
03 — Training

How it learns.

1

Tokenize the corpus

Raw text is encoded into integer token IDs using the BPE tokenizer trained on your corpus. Vocabulary size scales automatically with corpus size.

2

Sliding window batches

Token sequences are sliced into overlapping windows. Each window becomes an (input, target) pair where target is input shifted one position right.

3

Forward pass + cross-entropy loss

The model predicts the next token at every position simultaneously (teacher forcing). Loss is the mean cross-entropy across all positions.

4

AdamW + cosine LR schedule

Gradients are clipped to norm 1.0, then weights updated with AdamW. Learning rate follows a cosine decay schedule with a 10× warmdown.

5

Nucleus sampling at inference

Top-p (nucleus) sampling with temperature scaling. Adjust --top_p and --temperature to trade creativity for coherence.
